  • Dress up game

  • This project was based on JavaScript. A couple months before this I already learned the basics of HTML and CSS, so now it was time for some JavaScript. We were free to think of anything, as long as it manipulated the DOM. I had a few ideas, and I eventually decided I was going to make a dress up game. I made everything in Adobe Illustrator, added it to my site using HTML and positioned it using CSS. Then it was time to make sure everything didn’t show all at once, and then make it appear again at the right moments.
